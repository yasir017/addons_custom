Purchase Order Multi Product Selection.

This module allows you to add multiple products in purchase order line on a single click. You can also create request for quotation from product variant Tree view with selected multiple products very quickly.

Module Features
  • Purchase Order Multi Product Selection.
  • Add multiple lines to a purchase order line from purchase order at a time on single click .

Added one button in request for quotation view i.e "Select Product"

The wizard will open after click on "Select Product" button.

You will get the selection of product when you click on add an item.

Selected Product will be added in wizard.

All the selected line will be added in order lines if you will click on apply.

Select product as much as you want from the product variant tree view and click on action menu. There you will see a create request for quotation menu.

Wizard for create request for quotation will open.

RFQ will be created with selected products.
